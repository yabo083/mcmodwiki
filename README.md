# 机械动力电力学 Wiki

这是基于 Create-Electro-Energetics 的玩家向中文资料站。

从左侧目录进入各专题页面。

## 自动目录说明

- 侧边栏 `_sidebar.md` 由 GitHub Actions 自动生成。
- 新增或修改 Markdown 文章后，推送到 `main` 即可自动部署并更新目录。
- 支持按文件夹组织文档（会递归生成多级目录）。
- 如需本地预览自动目录，可运行：

```bash
python .github/scripts/generate_sidebar.py --root . --output _sidebar.md
```
